Explicación de las Pseudoclases

  • a:link — Estilo del enlace que aún no ha sido visitado.
  • a:visited — Estilo cuando el enlace ya fue visitado anteriormente.
  • a:hover — Estilo cuando el usuario pasa el mouse sobre el enlace.
  • a:focus — Estilo cuando el enlace tiene el foco (por teclado o tab).
  • a:active — Estilo en el momento exacto del clic sobre el enlace.

Nota importante: El orden correcto para aplicar estas pseudoclases es link → visited → hover → focus → active para evitar que unos estilos sobreescriban a otros (regla mnemotécnica: LoVe FocA).
